----------------------------------
What's That Color v0.3.1
----------------------------------
What's That Color is an app to get the current pixel color
at mouse pointer in RGB, HSB, HEX and CMYK values 
----------------------------------
HOW TO RUN:

to run the app just run the RUN.BAT file or
directly run the WTC.jar from the 'dist' folder
----------------------------------
HOW TO ADD ICC PROFILES

to add ICC profiles download the '.icc' files and put them in 
the 'ICC_PROFILES' folder. just restart the app to load the profiles.

the ICC profiles download link is provided in the 'ICC PROFILES LINKS.TXT' file.

to remove a profile just delete that profile file from the 'ICC_PROFILES' folder.
----------------------------------

Note: the user manual is present in the'DOCUMENTATION' folder
